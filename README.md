- Quest Avitar for Meta Quest 2 on Unity OpenXR
This project is a Unity VR starter environment for the Meta Quest 2 using OpenXR
This will be the foundation for a hands on VR avatar system with controller tracking and object interaction.
-- Current Features
  - Unity 2022 LTS project setup
  - Android build pipeline configured
  - OpenXR enabled
  - Oculus Touch controller profile active
  - Successfully deploys to Meta Quest 2
  - GitHub version control initialized
-- Tech Stack
  - Unity 2022 LTS
  - OpenXR Plugin
  - Meta Quest 2
  - Android Build Support
  - C#
-- Goal
This will be the starting point for a interactive VR avatar system using:
  - hand/controller tracking
  - object interaction
  - avatar embodiment
  - future AI-driven behavior integration

OpenXR environment setup is complete.
The next step will be to add interaction components and an avatar control system
